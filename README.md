# COVID‑19 Data Insights: Exploratory Analysis and Interactive Tableau Report

## 📌 Overview
This project demonstrates end-to-end exploratory data analysis (EDA) using **SQL** for data cleaning and preprocessing, and **Tableau** for interactive visualization.  
The workflow highlights how raw datasets can be transformed into clean, structured data and then visualized to uncover trends, correlations, and actionable insights.

---

## 🛠 Tech Stack
- **SQL** (MySQL/PostgreSQL/SQLite) → Data cleaning, profiling, and transformations  
- **Tableau Public** → Interactive dashboards and visual storytelling  
- **Excel/CSV datasets** → Raw input data  

---

## 📂 Project Structure
- `data/` → Raw and cleaned datasets (Excel/CSV)  
- `sql/` → SQL scripts for cleaning, aggregation, and preparing data for Tableau  
- `tableau/` → Tableau Public workbook (`.twbx`) and dashboard files  
- `figures/` → Screenshots of Tableau dashboards for quick preview  
- `README.md` → Project documentation  

---

## ⚙️ Workflow
1. **Data Cleaning (SQL)**
   - Handle missing values  
   - Remove duplicates  
   - Standardize formats  
   - Create analysis-ready tables  

2. **Exploratory Data Analysis (SQL)**
   - Generate descriptive statistics  
   - Profile datasets (counts, averages, correlations)  
   - Identify anomalies and outliers  

3. **Visualization (Tableau)**
   - Import cleaned dataset into Tableau  
   - Build dashboards: bar charts, line charts, maps, heatmaps  
   - Highlight trends, distributions, and correlations
